Help the researcher build a proper genealogical citation for: $ARGUMENTS

Load the `citation-guide` skill and `references/jones-citation-framework.md` from the `gps-methodology` skill.

**Do NOT generate the citation automatically.** The purpose of this command is to teach the researcher to think through citations themselves. Walk them through the Jones five-question framework interactively.

**Step 1 — Ask: "Who created this record?"**
Help them identify the author, compiler, informant, agency, or record office. If they're unsure, ask what kind of document it is and help them figure out who would have created it.

Common confusion points:
- The website is NOT the creator (Ancestry didn't create the census)
- The indexer is NOT the creator (unless you're citing the index itself)
- For vital records, also note the informant if visible

**Step 2 — Ask: "What is this source?"**
Help them name it. Published works have titles. Unpublished records need clear descriptions.

Common confusion points:
- "A record from Ancestry" is not a source name
- Use the record's own title, not the database name (unless citing the database itself)

**Step 3 — Ask: "When was this created?"**
Publication year for books. Event or recording date for records. Access date for online sources.

**Step 4 — Ask: "Where is this source held?"**
Repository name and location. Website URL. Publisher information.

**Step 5 — Ask: "Where exactly in this source did you find this information?"**
Page, volume, entry, certificate number, image number, line number — whatever makes the exact location findable.

**Step 6 — Assemble and review.**
Put the five answers together into a properly formatted citation. Show the researcher how each part maps to the five questions. Ask: "Does this look right? Could someone else find this exact record using this citation?"

**Step 7 — Classify the source.**
After the citation is built, briefly classify: original/derivative/authored, primary/secondary/undetermined information, direct/indirect/negative evidence. Explain what this means for how much weight the source carries.

If the researcher has a research plan or evidence log in their Google Drive folder, offer to add the citation there.
