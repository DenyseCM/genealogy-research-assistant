# Genealogical Citation Guide

Proper citation is GPS Element 2. Every piece of information used in genealogical research must be cited so that anyone can locate the same source and verify the claim.

## The Jones Five-Question Framework

Every citation must answer these five questions:

1. **Who** — Who created the source?
2. **What** — What is the source?
3. **When** — When was it created?
4. **Where is** — Where is the source held?
5. **Where in** — Where exactly in the source is the information?

See `references/jones-citation-framework.md` in the `gps-methodology` skill for the full framework with examples by source type.

## How to Coach Citation Building

When a researcher needs to cite a source, do NOT generate the citation for them automatically. Instead, walk them through the five questions:

1. Ask: "Who created this record? Was it a government agency, a church, a person?"
2. Ask: "What is this document called? If it doesn't have a title, describe what it is."
3. Ask: "When was this created or recorded?"
4. Ask: "Where did you find it? Which archive, website, or repository?"
5. Ask: "Where exactly in that source — what page, entry, or image number?"

After they answer all five, assemble the citation and show them how each part maps back. The goal is teaching the framework, not just producing citations.

## Citation Formats

### First (Full) Reference Note
Include all five elements in complete detail. This is used the first time a source appears.

### Subsequent (Short) Reference Note
After the first full citation, use a shortened form: Author/Creator, short title, and the specific location within the source.

### Source List Entry
Alphabetical list of all sources consulted. Organized by author/creator. Does not include the specific "where in" detail — that goes in reference notes only.

## Source Classification

When citing, also classify the source:

- **Original record**: Created at or near the time of the event by someone with knowledge of it
- **Derivative record**: A copy, transcription, abstract, or extract of an original
- **Authored narrative**: Someone's interpretation or compilation of information from multiple sources

This classification affects how much weight the source carries in evidence evaluation.

## Information Classification

Note who provided the information:

- **Primary information**: Provided by someone with firsthand knowledge of the event
- **Secondary information**: Provided by someone who learned about it from others
- **Undetermined**: Cannot tell who the informant was or their relationship to the event

## Quick Reference: Common Source Types

| Source Type | Who (typical) | What (example) | Where In (specificity needed) |
|-------------|--------------|----------------|------------------------------|
| Census | Bureau of the Census | 1870 U.S. census, population schedule | County, township, page, dwelling, family, line |
| Vital record | State/county health dept | Birth certificate | Certificate number |
| Church record | Church name, location | Baptismal register | Page, entry, date |
| Deed | County recorder | Deed Book [letter] | Book, page |
| Probate | County court | Will of [name] | File number, page |
| Newspaper | Paper name | Article title or "obituary of [name]" | Date, page, column |
| Online database | Database creator | Database title as displayed | Entry details, image number, access date |
| Cemetery | Cemetery name, location | Gravestone inscription | Section, lot, row if known |
