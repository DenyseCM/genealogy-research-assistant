# Genealogical Proof Standard (GPS)

The GPS is the professional standard for credible genealogical research. All research assistance, evaluation, and reporting must follow these five interdependent elements.

## The Five Elements

### 1. Reasonably Exhaustive Search

Search all sources that might provide information relevant to the research question. "Reasonably exhaustive" means:

- Check all record types that could contain relevant information for the time period and place
- Search both original records and derivative works
- Look in expected AND unexpected repositories
- Document negative searches (what was looked for but not found)
- A search is exhaustive enough when additional searching is unlikely to change the conclusion

Do not allow researchers to skip record types because they seem unlikely. The GPS requires looking where information *might* be, not just where it probably is.

### 2. Complete and Accurate Source Citations

Every piece of information must be cited so that anyone can find the same source and verify the claim. Use the Jones five-question framework:

1. **Who** created or authored the source
2. **What** the source is (title or description)
3. **When** it was created or published
4. **Where is** the source held (repository, website, publisher)
5. **Where in** the source the specific information appears (page, entry, image number)

See `references/jones-citation-framework.md` for the full citation guide with examples by record type.

### 3. Analysis and Correlation of Evidence

Analyze each source individually, then correlate information across sources:

- **Source classification**: Is this an original record, a derivative, or authored narrative?
- **Information classification**: Is the informant providing firsthand, secondhand, or undetermined information?
- **Evidence classification**: Is this direct evidence (explicitly states the fact), indirect evidence (requires inference), or negative evidence (expected information is absent)?
- **Correlation**: Compare what multiple sources say about the same fact. Do they agree? Disagree? Fill different gaps?

### 4. Resolution of Conflicting Evidence

When sources disagree, resolve the conflict — do not ignore it:

- Identify the specific conflict
- Evaluate each source's likely accuracy (who provided the information, when, under what circumstances)
- Determine which source is more likely reliable and explain why
- Account for the less-reliable source (explain why it may be wrong)
- If the conflict cannot be resolved, say so explicitly

### 5. Soundly Reasoned, Coherently Written Conclusion

The conclusion must:

- State the finding clearly
- Be supported by the evidence presented
- Address all conflicting evidence
- Follow logical reasoning (not leaps of faith)
- Be written so that another researcher can follow the logic and evaluate it independently

## Applying GPS in Practice

When helping a researcher, always track which GPS elements have been satisfied and which still need work. The standard is not optional — it is what separates credible research from guesswork.
